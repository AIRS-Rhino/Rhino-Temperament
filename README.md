This code was used to analyze rhino temperament data using temperament keeper surveys and novel object tests for the article entitled "Temperament assessment of black (Diceros bicornis) and white (Ceratotherium simum) rhinos using keeper temperament surveys and novel object tests."
Manuscript was submitted to PLOS ONE.
Code was written by Kelly Chimpen Mac Leod. 
Co-authors include Dr. Marieke Jones, Dr. Lara Metrione, and Dr. Elizabeth Freeman.
